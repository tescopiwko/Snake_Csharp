Toto je moje originalní verze hry Snake, doufám že vám hraní přinese spoustu zábavy, proto hrajte a užívejte si to :D
Jak hra funguje: 
    1. První had s oranžovou barvou se ovládá šipkami a druhý solidní had zelený se ovládá pomocí WASD.
    2. Barva pozadí je od začátku nastavena na světle modrou, ale mění se podle barvy hada, který naposledy zvítězil.
    3. Nahoře v popisu okna se také počítá skóre obou hadů, které se resetuje když hru vypnu a zapnu.
